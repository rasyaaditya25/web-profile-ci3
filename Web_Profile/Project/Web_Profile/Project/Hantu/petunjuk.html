Petunjuk Main Membasmi Hantu


1. Masukkan username Anda.

2. Pilih level permainan (Easy, Medium, Hard).

3. Klik 'Start Game!' untuk memulai permainan.

4. Klik pada hantu untuk mendapatkan skor.

5. Waktu akan berjalan, kumpulkan skor sebanyak-banyaknya sebelum waktu habis.

6. Setelah game over, Anda bisa menyimpan skor Anda di leaderboard.
